# Changelog

## 0.1.0

- Initial release
- `get()` with type-safe path inference and default values
- `set()` with immutable updates
- `has()` for path existence checking
- `del()` for immutable property removal
- `update()` for transform-in-place at a path
- Full `Paths<T>` and `PathValue<T, P>` type utilities
